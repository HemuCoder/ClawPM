# 主题索引

这个文件是主题记忆的总入口。

## 使用规则

- 每次会话启动时先读取本文件，了解当前有哪些可复用主题。
- 当用户提到明确项目/主题时，先看这里是否已有命中项。
- 命中后再读取对应 topic 文件。
- 新建主题后，要立刻在这里补一条索引。
- 主题结束或失效后，不要删历史；更新状态即可。

## 加载 checklist

1. 读取本文件
2. 识别用户当前提到的项目/主题/关键词
3. 命中后读取对应 topic 文件
4. 结合今天/昨天的日记继续工作
5. 结束后回写 topic 文件与当日日记
